# 📌 Project Overview

This repository contains an end-to-end **E-Commerce Sales Analysis** project developed using **Python** in **Google Colab**. It is based on the **roadmap.sh E-Commerce Data Analysis** project and extends it with additional business analytics, customer insights, and professional data visualizations.

The project follows a complete data analytics workflow:

* 📥 Data Collection
* 🧹 Data Cleaning & Preprocessing
* 📊 Exploratory Data Analysis (EDA)
* 💰 Revenue Analysis
* 🌍 Country-wise Sales Analysis
* 👥 Customer Analytics
* 📦 Product Performance Analysis
* 📈 Business KPI Dashboard
* 📉 Interactive Visualizations
* 💡 Business Recommendations

---

# 🎯 Project Objectives

* Analyze over **500,000 retail transactions** from the UCI Online Retail II dataset.
* Clean and preprocess transactional data.
* Remove cancelled invoices and invalid transactions.
* Create a **Revenue** feature for business analysis.
* Identify the **Top 10 Countries by Revenue**.
* Analyze monthly and hourly sales trends.
* Discover top-performing products and customers.
* Build business KPIs and visualizations.
* Develop a portfolio-ready analytics project following industry best practices.
